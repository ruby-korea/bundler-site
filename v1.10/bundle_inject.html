bundle inject

결정된 Gemfile에 gem 이름과 버전을 추가합니다. 
$ bundle inject [GEM] [VERSION]
gem을 주입할 때, Gemfile과 Gemfile.lock에 아직 없는 경우 양쪽에 해당 gem을 추가하게 됩니다. 밑의 예제를 보세요. 
$ bundle install

$ bundle inject 'rack' '> 0'
# 버전 0 이상의 rack gem을 Gemfile, Gemfile.lock에 주입
