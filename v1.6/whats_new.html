v1.6의 새로운 기능

이 단락에서는 이번 릴리스에 도입된 주요 기능을 확인할 수 있습니다. 모든 변경점은 번들러 1.6의 changelog에 문서화되어 있습니다. 1.6의 전체 changelog

재귀를 예방하고, 전보다 Gemfile 파일을 100배 빠르게 해결하도록 하기 위해 해결기를 다시 구현했습니다.

bundle console는 이제 Pry, Ripl 같은 대안 IRB를 지원합니다. Pry를 사용하고 싶다면 다음 번들러 console 설정을 하세요. bundle config console pry Gemfile 안에 pry가 있는지 확인 하셔야 합니다. 이제 bundle console을 실행하면 IRB 대신 Pry가 열립니다.
자세한 내용은: Bundle Console

이제 새로운 gem을 만들 때, bundle gem은 C 확장의 뼈대를 만드는 --ext 플래그를 지원합니다. bundle gem newgem --ext을 실행하면, ext 디렉터리 안에 extconf.rb, newgem.c, newgem.h 같은 새로운 파일을 만듭니다.
자세한 내용은: Bundle Gem

번들러 1.6은 다음 패치도 포함합니다.
  • 해결과 설치를 빠르게 하는 성능 개선
  • :github, :gist 같은 사용자 정의 옵션을 위한 :git_source
  • 설치되거나 업데이트된 gem의 색 하일라이트
  • HTTP 인증은 이제 bundle config에 저장됩니다.
