Command Line Reference

command applicable options
bundle binstubs
나열된 gem의 binstubs를 설치
--force
--path
bundle check
Gemfile의 목록이 현재 설치된 gem의 의존성을 만족하는지 확인
--dry-run
--gemfile
--path
bundle clean
번들러 디렉터리에서 사용하지 않는 gem을 청소
--dry-run
--force
bundle config
설정 값을 보거나 지정
bundle console
번들이 프리로드된 IRB 세션 열기
bundle exec
번들의 컨텍스트에서 명령어 실행
--keep-file-descriptors
bundle gem
루비젬을 만들기 위한 뼈대 생성
--bin or -b
--edit
--ext
--test
bundle help
각 하위 명령어의 자세한 도움말 표시
bundle init
현재 디렉터리에 Gemfile 생성
--gemspec
bundle inject
결정된 Gemfile에 gem 이름과 버전 추가
bundle install
시스템에 현재 환경을 설치
--binstubs
--clean
--deployment
--frozen
--full-index
--gemfile
--jobs
--local
--no-cache
--no-prune
--path
--quiet
--retry
--shebang
--standalone
--system
--trust-policy
--without
bundle open
주어진 번들 gem의 소스 디렉터리 열기
bundle outdated
새로운 버전이 나온 설치된 gem 목록
--local
--pre
--source
bundle package
vendor/cache에 모든 gem을 캐시, 락
--all
--gemfile
--no-install
--no-prune
--path
--quiet
bundle platform
플랫폼 호환성 정보 표시
--ruby
bundle show
번들에 포함된 모든 gem의 목록이나, 특정 gem의 경로를 표시
--paths
bundle update
현재 환경을 업데이트
--full-index
--group
--jobs
--local
--quiet
--source
bundle version
버전 정보 출력
bundle viz
비주얼 의존성 그래프 생성
--file or -f
--format or -F
--requirements or -r
--version or -v
