번들러를 레일즈 3과 사용하기

평범하게 레일즈를 설치하세요. 평소에 젬을 설치할때 sudo로 하시면 sudo도 사용해 주세요. 
$ gem install rails
다른 루비버전들을 사용하실 일이 있다면, 신뢰할 수 있는 루비 설치관리를 위해 rvm을 추천합니다.
평소처럼 레일즈앱을 생성합니다. 
$ rails new myapp
$ cd myapp
서버를 실행합니다. 이미 번들러는 의존성들을 투명하게 관리 하고있습니다! 
$ rails server
필요한 새 의존성을 Gemfile에 추가합니다. 
gem 'nokogiri'
gem 'geokit'
특정 레일즈 환경에서만 로드하고싶은 의존성들이 있다면, 레일즈 환경을 이름으로 한 그룹에 넣으세요. 
group :test do
  gem 'rspec'
  gem 'faker'
end
한번에 여러 그룹에 의존성을 넣을 수도 있습니다. 
group :development, :test do
  gem 'wirble'
  gem 'ruby-debug'
end
자세한 내용은: 그룹
의존성을 추가하고 아직 설치하지 않았다면 설치하세요. 
$ bundle install
이 명령은 Gemfile 안의 모든 의존성들을 다른 의존성과 충돌하지 않는 최신버전으로 갱신합니다.
